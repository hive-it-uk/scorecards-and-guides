🛡️ Coding-AI governance & code-sovereignty rules

  • • Keep local references clean: Don't hard-code API passwords, client credentials, or private connection strings into workspaces where AI autocomplete runs. Use environment variables.
  • • Audit editor telemetry settings: Most IDE plugins log context chunks. Confirm in your global configuration that telemetry sends are restricted.
  • • Use commercial / enterprise accounts for sensitive work: If you process IP-sensitive codebases or client-protected modules, use Enterprise or Team tiers. Free and personal subscriptions increasingly default to logging context for training.
  • • Configure ignore files: Use .copilotignore / .gitignore and equivalent settings to tell code-indexing tools which local files should never be parsed.
